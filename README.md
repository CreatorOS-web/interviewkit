# InterviewKit

InterviewKit is a collection of practical, free job-search tools designed to speed up resumes, applications, and interviews.

All tools generate structured prompts you can paste into ChatGPT and then customize.

3. Build a structured prompt  
4. Paste into ChatGPT  
5. Copy the result into your application

No account required.
